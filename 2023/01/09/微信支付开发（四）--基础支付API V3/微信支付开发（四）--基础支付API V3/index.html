为通知失败，微信会通过一定的策略定期重新发起通知，尽可能提高通知的成功率，但微信不保
证通知最终能成功。（通知频率为
15s/15s/30s/3m/10m/20m/30m/30m/30m/60m/3h/3h/3h/6h/6h - 总计 24h4m）
